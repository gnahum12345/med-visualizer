# NeuroCardio MedVisualizer

A web-based visualization tool for exploring molecular structures of essential neurological and cardiovascular medications.

## Features

- **Interactive 3D Molecules**: View and rotate molecular structures using 3Dmol.js.
- **Category Filtering**: Filter medications by Neurological or Cardiovascular categories.
- **Search**: Quickly find specific medications.
- **Detailed Information**: Click on a medication to see dosage, side effects, and mechanism of action.

## Technologies

- HTML5, CSS3, JavaScript
- [3Dmol.js](https://3dmol.csb.pitt.edu/) for molecular visualization
- [jQuery](https://jquery.com/)
- Google Fonts (Outfit)

## Setup

To run this project locally, simply open `index.html` in your web browser.

## Deployment

This project is ready to be hosted on GitHub Pages.
